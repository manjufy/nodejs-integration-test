# Nodejs Integration Tests Sample

### Run

```
npm run start
http://localhost:3000/api
http://localhost:3000/api/teams
http://localhost:3000/api/drivers
http://localhost:3000/api/races
```

### Run Tests

```
npm install -g mocha  // install mocha globally
npm test
```
